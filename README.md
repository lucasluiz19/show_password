# Visualizador de Senhas
Este é um simples aplicativo que permite ao usuário visualizar a senha digitada em um campo de entrada do tipo "password".

## Como usar
Para usar o aplicativo, siga os passos abaixo:

1- Abra o arquivo index.html no seu navegador.
2- Digite uma senha no campo de entrada.
3- Clique no botão do olho "Visualizar senha".
A senha digitada será exibida no campo do input.

## Tecnologias utilizadas
O aplicativo foi criado usando as seguintes tecnologias:

* HTML
* CSS
* JavaScript

## Funcionamento
O aplicativo utiliza JavaScript para alterar o tipo do campo de entrada de "password" para "text", permitindo que a senha digitada seja visualizada. Ao clicar novamente no botão "Visualizar senha", o tipo do campo de entrada é revertido para "password", ocultando novamente a senha.
